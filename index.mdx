The Things Stack is an open source LoRaWAN Network Server designed for large, global, and geo-distributed deployments. It brings together the Network Server, Application Server, Join Server, and Gateway Server into one cohesive platform — giving you full control over your IoT infrastructure from device registration to application data delivery.

## What is The Things Stack?

The Things Stack implements the LoRaWAN protocol stack in full — from radio scheduling in the Gateway Server through MAC-layer processing in the Network Server, all the way to payload decoding and application delivery in the Application Server. It supports every major LoRaWAN version and device class, making it suitable for both small private networks and large-scale public deployments.

<Steps>
  <Step title="Install and configure">
    Deploy The Things Stack with Docker Compose. Configure your frequency plan, TLS certificates, and database connections.
  </Step>
  <Step title="Connect a gateway">
    Register a LoRaWAN gateway using the Console or CLI. Gateways communicate with the Gateway Server over UDP, BasicStation, or proprietary protocols.
  </Step>
  <Step title="Register an end device">
    Add your IoT device to an application. Choose OTAA for secure over-the-air activation or ABP for static session keys.
  </Step>
  <Step title="Receive and process data">
    Subscribe to your application data via MQTT, configure a webhook to push events to your backend, or use the storage integration for historical data access.
  </Step>
</Steps>
